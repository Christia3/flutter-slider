# Flutter Slider Widget Demo

## 📱 Description

This app demonstrates the Flutter Slider widget using a simple brightness control example.

## ▶️ How to Run

1. Clone the repository
2. Open the project in VS Code or Android Studio
3. Run the following commands:
   flutter pub get
   flutter run

## 🎯 Widget Properties Demonstrated

* **value**: Controls the current position of the slider and updates the brightness level.
* **min & max**: Define the range of the slider (0 to 100).
* **activeColor**: Changes the color of the slider track.
